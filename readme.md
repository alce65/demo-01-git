# Demo 01

Esto es una demo de cómo usar git, empezando con la creación de un repositorio.

Añadimos contenido, lo pasamos a la staged area y  realizamos el primer commit.

Hablamos sobre mensajes de los commits: deben contar el porque, no lo que contienen, y deben ser cortos y concisos.

Conectamos con un remote en GitHub y lo usamos para editar

Hemos añadido ramas y hemos hecho merge de una rama en otra.

Ahora empezamos a trabajar con los remotos.

Más adelante veremos las PR

Comprobamos que pull es la suma de fetch y merge
